# PrestaShop on Docker

Run

```
docker-compose up -d
```

and open <http://localhost/>

* [PrestaShop](http://127.0.0.1:8000/)
* [phpmyadmin](http://127.0.0.1:8181/)

На дефолте не работает под Windows.
